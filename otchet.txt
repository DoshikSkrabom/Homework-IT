Отчёт по выполнению практической работы №12
Шаги выполнения практической работы: 
1. Установка GIT
2. Создание локального репозитория с помощью команды git init 
3. Создание отчёта по выполнению практической работы и помещение его в репозиторий с помощью команды echo и git add
4. Фиксация всех проделанных шагов в отчёте и создание версии в GIT с помощью команды git commit -m
5. Создание ветки отчета в GIT, добавление описания шагов и создание отчёта в новой ветке
6. Добавление последнего изменения отчета в основную ветку
7. Регистрация на github
8. Создание проекта Homework-IIT, добавление содержимого из локального репозитория на github
9. Размещение на github раннее созданного статического сайта
10. Завершение отчёта
